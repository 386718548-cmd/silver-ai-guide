## 🚀 快速开始指南

### 本地运行
```bash
npm install      # 安装依赖
npm run dev      # 启动开发服务器
npm run build    # 构建生产版本
npm run lint     # 代码检查
```

### 部署到 GitHub Pages
```bash
npm run deploy   # 自动构建 + 部署
```

### 访问应用
```
本地：http://localhost:8080/
在线：https://YOUR_USERNAME.github.io/silver-ai-guide/
```
