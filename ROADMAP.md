## 🎤 Phase 2: 语音朗读功能（预计 4 月）

### 功能描述
为整个应用添加完整的语音朗读支持，让用户"听"而不仅仅"看"。

### 具体实现
- [ ] 在每个课程页面添加 **一键朗读** 按钮
- [ ] 集成 Web Speech API（支持中文）
- [ ] 添加 **语速控制**（快/正常/慢）
- [ ] 支持暂停/继续/停止
- [ ] 朗读状态提示（加载中、进行中、完成）

### 代码示例
```typescript
const speak = (text: string) => {
  const utterance = new SpeechSynthesisUtterance(text);
  utterance.lang = 'zh-CN';
  utterance.rate = 0.8; // 放慢速度
  window.speechSynthesis.speak(utterance);
};
```

### 优点
✅ 减少阅读压力  
✅ 适合视力不适的用户  
✅ 提高学习效率  

### 浏览器兼容性
- ✅ Chrome/Edge (完美支持)
- ✅ Firefox (完美支持)
- ✅ Safari (支持，中文语音略差)
- ⚠️ 某些手机浏览器可能不支持

### 功能描述
在教程中嵌入视频讲解，提高学习效果。

### 具体实现

- [ ] 集成 B 站 iframe 播放器

- 📺 B 站（Bilibili）- 中文内容丰富

### 数据结构示例

```typescript
  content: "点击下方视频查看详细演示"
}
```

## 👨‍👩‍👧 Phase 5: 子女模式（预计 7 月）

### 功能描述
允许子女为父母推送内容、监督进度、设置提醒。

### 具体实现
- [ ] 创建家庭账户系统
- [ ] 子女端：后台管理系统
- [ ] 推送新课程到父母
- [ ] 查看父母的学习进度
- [ ] 设置定时提醒（"爸爸，今天该看课程了~"）

### 功能细节
| 功能 | 描述 |
|-----|------|
| 📱 设备配对 | 通过二维码链接父子账户 |
| 📊 进度追踪 | 子女可看到父母学完了哪些课 |
| 💬 消息提醒 | 推送友好的学习提醒 |
| 🎓 课程推送 | 子女可添加自定义课程 |
| 🔒 隐私保护 | 只能看进度，不能看详细行为 |

### 示例场景
👩‍🦱 女儿给父亲推送"防诈骗"课程  
📲 系统自动提醒："爸爸，女儿给您推荐了新课，去看看吧"  
✅ 父亲看完后，女儿收到通知

## 🎓 Phase 6: 互动练习和测验（预计 8 月）

### 功能描述
为每个课程添加边学边练的小测验，巩固学习效果。

### 具体实现
- [ ] 每课 3-5 个选择题
- [ ] 闯关模式（完成一课才能看下一课）
- [ ] 获得徽章和成就系统
- [ ] 记录错题库，复习时特别强调
- [ ] 学习报告（学完时间、准确率等）

### 题目示例
```
Q: 如果接到子女的视频通话要求转账，最安全的做法是？
A) 直接转账
B) 先挂断，用自己的号码回拨原号码确认
C) 问对方密码
D) 不理他

✅ 正确答案：B
💡 解释：这是防止 AI 换脸诈骗的最保险方法
```
