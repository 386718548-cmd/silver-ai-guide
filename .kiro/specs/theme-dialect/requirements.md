# 需求文档

## 简介

本文档描述"银发 AI 指南"项目新增的两个功能模块：

1. **UI 主题系列切换**：让用户根据感官偏好选择界面风格，提供自然草本、中华传统、现代简约、水墨养生四套主题，全局切换并持久化保存。
2. **方言朗读功能**：在现有普通话语音朗读基础上，扩展支持粤语、闽南语、四川话、上海话、东北话等中文方言，优先使用 Web Speech API，不可用时降级至第三方 TTS 服务（如讯飞、百度语音）。

目标用户为 55 岁以上中老年人，界面设计须保持适老化标准。

---

## 词汇表

- **主题系统（ThemeSystem）**：管理和应用 UI 主题的模块，负责主题切换、持久化和全局样式注入
- **主题（Theme）**：一套完整的 CSS 变量配置，包含背景色、主色调、文字色、边框色等视觉参数
- **方言朗读引擎（DialectEngine）**：扩展现有语音引擎，支持多种中文方言的 TTS 朗读
- **Web Speech API**：浏览器原生语音合成接口
- **TTS 服务（TTSService）**：第三方文字转语音 API，用于 Web Speech API 不支持的方言
- **语音引擎（SpeechEngine）**：现有基于 Web Speech API 的语音朗读模块（已有功能）
- **设置页面（SettingsPage）**：用户偏好设置界面，包含主题选择和方言选择
- **用户（User）**：55 岁以上的中老年学习者

---

## 需求

### 需求 1：主题系列定义

**用户故事：** 作为一位中老年用户，我希望应用提供多种视觉风格，以便我能选择最符合自己审美和视觉习惯的界面。

#### 验收标准

1. THE 主题系统 SHALL 提供以下四套预设主题：
   - **自然草本风（nature）**：主色调为绿色（#4a7c59），背景为米白色（#f5f0e8），文字为深棕色（#2d2416），边框为浅绿色（#c8dcc0）
   - **中华传统风（chinese-classic）**：主色调为朱红色（#c0392b），背景为暖米色（#fdf6ec），文字为深墨色（#1a0a00），边框为金棕色（#d4a853）
   - **现代简约风（modern-clean）**：主色调为蓝绿色（#0891b2），背景为纯白色（#ffffff），文字为深灰色（#1a1a1a），边框为浅灰色（#d1d5db）
   - **水墨养生风（ink-wellness）**：主色调为深墨色（#2c3e50），背景为牛皮纸色（#f4ede0），文字为深棕色（#2c1810），边框为浅棕色（#c9b99a）
2. THE 主题系统 SHALL 为每套主题提供中文名称、英文标识符、代表色预览和一句话风格描述。
3. THE 主题系统 SHALL 确保所有主题的文字与背景颜色对比度不低于 WCAG AA 标准（4.5:1）。

---

### 需求 2：主题切换交互

**用户故事：** 作为一位中老年用户，我希望能在设置页面直观地预览并选择主题，以便我能看到效果后再决定。

#### 验收标准

1. THE 设置页面 SHALL 以卡片形式展示全部四套主题，每张卡片包含主题名称、色块预览（不少于 3 种代表色）和风格描述。
2. WHEN 用户点击某一主题卡片，THE 主题系统 SHALL 在 300 毫秒内将全局 CSS 变量切换为该主题的配置。
3. WHEN 主题切换完成，THE 设置页面 SHALL 在当前选中主题卡片上显示"✅ 当前主题"标记。
4. THE 设置页面 SHALL 为每张主题卡片提供不低于 120×80 像素的可点击区域，满足适老化触控要求。
5. WHEN 用户切换主题，THE 主题系统 SHALL 立即更新页面所有组件的视觉样式，无需刷新页面。

---

### 需求 3：主题持久化

**用户故事：** 作为一位中老年用户，我希望我选择的主题在下次打开应用时仍然生效，以便我不需要每次重新设置。

#### 验收标准

1. WHEN 用户选择一个主题，THE 主题系统 SHALL 将主题标识符保存到浏览器 localStorage，键名为 `silver-ai-theme`。
2. WHEN 用户再次打开应用，THE 主题系统 SHALL 在页面渲染前从 localStorage 读取已保存的主题并应用，避免出现主题闪烁（FOUC）。
3. IF localStorage 中无主题记录或记录值无效，THEN THE 主题系统 SHALL 默认使用"现代简约风（modern-clean）"主题。
4. THE 主题系统 SHALL 仅使用浏览器本地存储，不向任何服务器上传用户主题偏好数据。

---

### 需求 4：方言选择配置

**用户故事：** 作为一位中老年用户，我希望能选择用我熟悉的方言朗读课程内容，以便我能更轻松地理解学习材料。

#### 验收标准

1. THE 设置页面 SHALL 提供方言选择列表，包含以下选项：普通话（默认）、粤语（广东话）、闽南语（台湾话/福建话）、四川话、上海话、东北话，共 6 种。
2. THE 设置页面 SHALL 为每种方言显示中文名称和括号内的别称（如"粤语（广东话）"）。
3. WHEN 用户选择一种方言，THE 设置页面 SHALL 将选中方言标识符保存到 localStorage，键名为 `silver-ai-dialect`。
4. WHEN 用户再次打开应用，THE 方言朗读引擎 SHALL 从 localStorage 读取已保存的方言设置并应用。
5. IF localStorage 中无方言记录，THEN THE 方言朗读引擎 SHALL 默认使用普通话（zh-CN）。

---

### 需求 5：Web Speech API 方言朗读

**用户故事：** 作为一位中老年用户，我希望应用优先使用浏览器内置语音朗读方言，以便不依赖网络也能使用。

#### 验收标准

1. WHEN 用户触发朗读且选择的方言为普通话，THE 方言朗读引擎 SHALL 使用 Web Speech API 以 `zh-CN` 语言代码朗读。
2. WHEN 用户触发朗读且选择的方言为粤语，THE 方言朗读引擎 SHALL 优先尝试使用 Web Speech API 的 `zh-HK` 或 `yue` 语音包朗读。
3. WHEN 用户触发朗读且选择的方言为闽南语，THE 方言朗读引擎 SHALL 优先尝试使用 Web Speech API 的 `zh-TW` 语音包朗读。
4. WHEN 用户触发朗读且选择的方言为四川话、上海话或东北话，THE 方言朗读引擎 SHALL 检测 Web Speech API 是否有对应语音包可用。
5. IF Web Speech API 不支持用户选择的方言，THEN THE 方言朗读引擎 SHALL 降级至第三方 TTS 服务，并在朗读前提示用户"正在使用在线语音服务朗读"。
6. IF Web Speech API 和第三方 TTS 服务均不可用，THEN THE 方言朗读引擎 SHALL 回退至普通话朗读，并提示用户"当前方言暂不可用，已切换为普通话朗读"。

---

### 需求 6：第三方 TTS 服务集成

**用户故事：** 作为一位中老年用户，我希望在浏览器不支持方言时，应用能通过在线服务提供方言朗读，以便我仍能享受方言朗读功能。

#### 验收标准

1. THE 方言朗读引擎 SHALL 支持配置第三方 TTS API 接入点（如讯飞 TTS 或百度语音），通过环境变量注入 API 密钥，不在代码中硬编码。
2. WHEN 调用第三方 TTS 服务，THE 方言朗读引擎 SHALL 在 5 秒内开始播放音频；IF 超时，THEN THE 方言朗读引擎 SHALL 显示错误提示并停止朗读。
3. WHEN 第三方 TTS 服务返回错误，THE 方言朗读引擎 SHALL 显示提示"语音服务暂时不可用，请稍后重试"，并停止朗读流程。
4. THE 方言朗读引擎 SHALL 在调用第三方 TTS 服务前，向用户展示"正在连接语音服务..."的加载状态。
5. WHERE 第三方 TTS API 密钥未配置，THE 方言朗读引擎 SHALL 跳过第三方服务，直接回退至普通话朗读，不抛出未处理的异常。

---

### 需求 7：朗读状态与控制

**用户故事：** 作为一位中老年用户，我希望在方言朗读时能清楚看到当前状态，并能随时停止，以便我掌控朗读过程。

#### 验收标准

1. WHEN 方言朗读进行中，THE 方言朗读引擎 SHALL 在朗读按钮旁显示当前使用的方言名称（如"正在用粤语朗读..."）。
2. WHEN 用户点击停止按钮，THE 方言朗读引擎 SHALL 立即停止所有朗读（包括 Web Speech API 和第三方 TTS 音频）。
3. WHEN 用户在朗读进行中切换方言，THE 方言朗读引擎 SHALL 先停止当前朗读，再以新方言重新开始朗读。
4. WHEN 用户离开当前课程页面，THE 方言朗读引擎 SHALL 自动停止正在进行的朗读。
5. THE 方言朗读引擎 SHALL 继承现有语音引擎的朗读速度设置（不高于正常语速的 80%）。

---

### 需求 8：设置页面入口与导航

**用户故事：** 作为一位中老年用户，我希望能方便地找到设置页面，以便我能随时调整主题和方言偏好。

#### 验收标准

1. THE 系统 SHALL 在应用首页的显眼位置（如页眉或导航栏）提供"⚙️ 设置"入口按钮。
2. WHEN 用户点击"设置"按钮，THE 设置页面 SHALL 在 300 毫秒内打开，展示主题选择和方言选择两个功能区。
3. THE 设置页面 SHALL 将主题选择区和方言选择区分别标注清晰的区域标题（如"🎨 界面主题"和"🗣️ 朗读方言"）。
4. THE 设置页面 SHALL 提供明显的"关闭"或"返回"按钮，按钮文字标注"返回"，触控区域不低于 44×44 像素。
5. WHEN 用户关闭设置页面，THE 系统 SHALL 返回用户进入设置前所在的页面，不丢失页面状态。
